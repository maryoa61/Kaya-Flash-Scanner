# ⚡ Kaya Flash Scanner Pro

یک اسکنر و فیلترکننده وب‌سرویس و لایو شبکه (Client-Side) با سرعت برق‌آسا (Flash Mode) که برای غربالگری رنج‌های متغیر و آی‌پی‌های ثابت کلودفلر طراحی شده است. این ابزار به طور هوشمند تمام آی‌پی‌های با پینگ واقعی **زیر ۱۵۰ میلی‌ثانیه** را صید کرده و مستقیماً به هسته مرکزی پنل شما (ورکر کلودفلر) شلیک می‌کند.

---

## ✨ قابلیت‌های کلیدی نسخه Pro

*   **ورودی دوگانه موازی (Dual Input):** دارای دو کادر ورودی کاملاً مجزا؛ یکی برای رنج‌های آی‌پی (`CIDR`) و دیگری برای لیست آی‌پی‌های تکی و ثابت.
*   **فیلترینگ سخت‌گیرانه پینگ (Under 150ms):** رد کردن تمام آی‌پی‌های کند و گلچین کردن لایو شبکه‌های با کیفیت بالا.
*   **دو جدول خروجی تفکیک‌شده:** جداسازی هوشمند نتایج رنج‌ها (`#KAYA_Range`) از آی‌پی‌های ثابت (`#KAYA_Single`).
*   **تشخیص خودکار اپراتور (ISP Detection):** تشخیص هوشمند شبکه شما (ایرانسل، همراه اول، مخابرات و شاتل) و زدن برچسب اختصاصی روی کانفیگ‌ها.
*   **کنترل کپی همه‌جانبه:** قابلیت کپی به صورت تکی، کپی جمعیِ تفکیک‌شده هر جدول و کپی یک‌جای کل دیتای سابسکریپشن ترکیبی (Base64).
*   **همگام‌سازی ابری خودکار:** اتصال مستقیم به ورکر پروژه اصلی (`Kaya Cloud Panel`) و تزریق خودکار بدون نیاز به کپی دستی.

---

## 🚀 نحوه راه‌اندازی و استفاده

1. این ریپازیتوری را روی گیت‌هاب خود ستاپ و از بخش **Settings > Pages** آن را آنلاین کنید.
2. آدرس سایت گیت‌هاب پیجز خود را باز کنید.
3. در بخش تنظیمات ابری، آدرس ورکر کلودفلر خود (بدون پسوند `/sub`) و رمز امنیتی (`Token`) را وارد کنید.
4. کانفیگ مادر (VLESS یا Trojan) خود را در کادر مربوطه قرار دهید.
5. رنج‌ها یا آی‌پی‌های موردنظر را وارد کرده و دکمه **⚡ شروع فیلترینگ و اسکن دوگانه** را بزنید.
6. تمام! آی‌پی‌های تمیز زیر ۱۵۰ms به طور خودکار به سابسکریپشن شما تزریق می‌شوند و در نرم‌افزارهای گوشی یا سیستم‌تان آپدیت خواهند شد.
